# simple-restapi

### Info
In this repository will demostrate how to build REST API's in Java as programming language and Spring Boot as a framework. 

This repository is using:
1. Java Vesrtion 11
2. Spring Boot 2.7
3. Lombok library
4. JpaData
5. Docker
